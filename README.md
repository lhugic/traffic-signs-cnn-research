# Исследование архитектуры сверточных нейронных сетей (CNN): Распознавание дорожных знаков

[СКАЧАТЬ ПОЛНЫЙ ТЕКСТ ИССЛЕДОВАНИЯ (PDF)](Traffic_Signs_CNN_Research.pdf)

## Суть работы
Данный репозиторий содержит академическую курсовую работу, посвященную глубокому теоретическому и математическому разбору алгоритмов компьютерного зрения. 

**Важное уточнение:** Программная реализация модели базируется на открытом решении с платформы Kaggle. Главная ценность данного проекта заключается не в написании кода с нуля, а в подробной декомпозиции модели и анализе математического аппарата нейронных сетей.

## Ключевые темы исследования
В приложенной работе (PDF) детально разобраны следующие аспекты машинного обучения:
* **Математическая модель CNN:** Принципы работы сверточных слоев и пулинга.
* **Оптимизация и метрики:** Анализ работы функции потерь и математики оптимизатора Adam. Обоснование выбора метрик оценки качества.
* **Работа с данными:** Проблема переобучения и применение методов аугментации для повышения робастности модели.

## Внутри репозитория
* `Traffic_Signs_CNN_Research.pdf` - полный текст исследовательской работы (теоретическая база, формулы, анализ). Программный код в приложении (в конце работы) снабжен подробными построчными комментариями.
